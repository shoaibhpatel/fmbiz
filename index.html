📤 Export Backup

Saves all your data (customers, projects, invoices, payments, expenses, settings) into a single JSON file.

Click "Export Backup" — it will ask you to select your StudioBusinessManager app folder once. After that, every backup auto-saves to the backup/ folder inside it, overwriting the same file each time.

📥 Import Backup

⚠️ This will replace all current data with the data inside the backup file.

🗑️ Reset System

Erase all data and start fresh. Cannot be undone.

⚙️ GitHub Sync Setup

📘 First-time Setup — 5 minute guide

Step 1. GitHub account banao (free, lifelong).

Step 2. Naya repo banao — naam kuch bhi (e.g. fmbiz). Public rakho (free GitHub Pages ke liye).

Step 3. Is app ki saari files (index.html, app.js, style.css, cloud-sync.js, DP logo.jpg, data/ folder) us repo me upload kar do — drag & drop se bhi ho jaata hai.

Step 4. Repo Settings → Pages → Source = Deploy from a branch → Branch = main / root → Save. 1-2 minute me URL milega: https://USERNAME.github.io/fmbiz/

Step 5. Personal Access Token banao:

  • 👉 github.com/settings/tokens?type=beta — "Generate new token" (Fine-grained)
  • Repository access → "Only select repositories" → apna fmbiz repo choose karo
  • Permissions → Repository permissions → Contents = Read and write
  • Expiration: 1 year (renew karte rehna)
  • Token copy karke neeche field me paste karo (sirf ek baar dikhega!)

Step 6. Upar form me Username, Repo, Token daal ke Save & Connect dabao. Bas!

💡 Tip: Token har device pe alag se daalna padega (security ke liye). Username + Repo + Token = setup complete. Phir wo device hamesha ke liye sync me rahega.
